# IVaultV2

The main interface for Morpho Vault V2, extending the ERC4626 tokenized vault standard and ERC2612 permit functionality.

```solidity
interface IVaultV2 is IERC4626, IERC2612
```

## Structs

### Caps

Defines allocation and cap constraints for adapters.

```solidity
struct Caps {
    uint256 allocation;
    uint128 absoluteCap;
    uint128 relativeCap;
}
```

<ParamField name="allocation" type="uint256">
  Current allocation amount for the adapter
</ParamField>

<ParamField name="absoluteCap" type="uint128">
  Maximum absolute allocation cap in asset units
</ParamField>

<ParamField name="relativeCap" type="uint128">
  Maximum relative allocation cap as a percentage of total assets
</ParamField>
